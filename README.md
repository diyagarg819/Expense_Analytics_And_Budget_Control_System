# Expense_Analytics_And_Budget_Control_System
A web-based personal finance management system that tracks expenses, manages budgets, supports vault accounts, and provides financial analytics through an interactive dashboard.

## Features
- User authentication (Signup & Login)
- Vault-based account management
- Expense tracking with category selection
- Transaction log with month, year, and category filters
- Monthly budget management for categories
- Savings goal tracking
- Lending and borrowing tracker
- Financial dashboard with analytics charts
- Dark / Light theme support
